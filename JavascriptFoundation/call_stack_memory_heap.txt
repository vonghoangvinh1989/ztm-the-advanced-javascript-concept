Javascript: CallStack and Memory Heap inside

1) Call Stack
    - where the engine keeps track of where your code is in its execution
    - region in memory to operate first in last out (FILO) mode
    - store functions and variables when the code executes

2) Memory Heap
    - memory allocation happens
    - store and write information
